# Fashion Game? 🎮

Интерактивная игра на Python с графическим интерфейсом, которая проверит, насколько хорошо ты разбираешься в моде! Отвечай на вопросы, делай выбор и узнай, кто ты: настоящий ИГРОК или ЛЭЙМ.

## 📋 Описание

Игра состоит из серии вопросов о моде и стиле. Каждый ответ приносит определенное количество баллов. В конце игры ты получишь свой результат и узнаешь, насколько хорошо ты шаришь в трендах.

## 🎯 Особенности

- 3 увлекательных вопроса о моде
- Разные варианты ответов с разным количеством баллов
- Стильный темный интерфейс
- Мгновенная реакция на твой выбор
- Финальная оценка с персональным описанием
- Возможность перезапустить игру

## 🛠 Требования

- Python 3.6 или выше
- PyQt5

## 📦 Установка

1. Клонируй репозиторий:
```bash
git clone https://github.com/Fram1se/fashion-game.git
cd fashion-game
```

2. Установи необходимые зависимости:
```bash
pip install PyQt5
```

3. Запусти игру:
```bash
python game.py
```

## 🎮 Как играть

1. Тебе будет показан вопрос с двумя вариантами ответов
2. Выбери один из вариантов, нажав на соответствующую кнопку
3. Получи мгновенную реакцию на свой выбор
4. После всех вопросов узнай свой результат

## 📊 Система подсчета очков

| Результат | Баллы | Описание |
|-----------|-------|----------|
| 👑 Король стиля | 12+ | Ты настоящий король стиля! |
| 🔥 Игрок | 8-11 | Неплохо, бро! Ты в теме |
| 🤔 Средний | 5-7 | Нужно больше разбираться |
| 💀 Лэйм | <5 | Пора смотреть модные блоги |

## 🎨 Интерфейс

- **Темная тема** - стильный черный фон
- **Золотой заголовок** - главный акцент
- **Крупные эмодзи** - визуальное отображение категорий
- **Анимированные кнопки** - приятная обратная связь
- **Цветовая индикация** - зеленый для хороших результатов, красный для плохих

## 🔧 Настройка

Ты можешь легко изменить игру:
- Добавить новые вопросы в массив `questions`
- Изменить баллы за ответы
- Настроить цвета интерфейса в `setStyleSheet`
- Поменять текст реакций
